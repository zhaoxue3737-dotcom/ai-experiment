# 实验页面使用说明

## 文件清单
- advisory.html   →  建议型AI组（被试自己选择商品）
- agentic.html    →  委托型AI组（AI替被试选择并下单）

---

## 核心操纵差异对照

| 节点 | 建议型（advisory.html） | 委托型（agentic.html） |
|------|----------------------|---------------------|
| AI回复 | 提供3个候选选项 + 推荐理由 | 显示"分析中"进度条，无选项 |
| 决策节点 | 被试点击选择一款商品 | AI直接展示"已为您选定" |
| 操作按钮 | 选择商品 + 确认下单（两步） | 只有"接受"或"取消"（无选项） |
| 完成码 | ADV-2024 | AGT-2024（取消则为AGT-CNCL） |
| 标题角标 | 蓝色"建议型" | 橙色"委托型" |

## 部署到 GitHub Pages（完全免费，5分钟完成）

### 第一步：注册 GitHub 账号
访问 github.com，注册免费账号。

### 第二步：创建仓库
1. 点击右上角 "+" → "New repository"
2. Repository name 填：ai-experiment
3. 勾选 "Public"
4. 勾选 "Add a README file"
5. 点击 "Create repository"

### 第三步：上传文件
1. 在仓库页面点击 "Add file" → "Upload files"
2. 把 advisory.html 和 agentic.html 拖进去
3. 点击 "Commit changes"

### 第四步：开启 Pages
1. 点击仓库的 "Settings"（右上方）
2. 左侧菜单找 "Pages"
3. Source 选 "Deploy from a branch"
4. Branch 选 "main"，文件夹选 "/ (root)"
5. 点击 "Save"
6. 等待约1分钟，页面会显示你的网址：
   https://你的用户名.github.io/ai-experiment/

### 第五步：得到两个实验链接
- 建议型链接：https://你的用户名.github.io/ai-experiment/advisory.html
- 委托型链接：https://你的用户名.github.io/ai-experiment/agentic.html

## 在 Credamo 中配置随机分组

### 问卷结构（在Credamo里按此顺序设置）

第1页：知情同意书
↓
第2页：分组跳转（Credamo"随机分组"功能）
  - A组（50%）→ 展示建议型链接
  - B组（50%）→ 展示委托型链接
↓
第3页：提示语（内嵌链接）
  内容示例：
  "请点击以下链接，完成一次AI购物模拟体验（约2分钟）。
   完成后请将页面显示的4位完成码填入下方输入框。"
  [点击此处开始体验] → 链接到对应的HTML页面
↓
第4页：填写完成码（验证被试确实完成了操纵）
  题型：填空题
  "请输入您在体验页面获得的完成码："
↓
第5页：操纵检验题（2题）
  MC1: 在刚才的操作中，最终购买决策由谁完成？
    A. 完全由我自己决定
    B. 主要由我决定，AI提供参考
    C. 主要由AI决定
    D. 完全由AI替我决定
  MC2: 我刚才使用的AI类型是？
    A. 建议型（给我提供多个选项，由我自己选）
    B. 委托型（替我做决定并下单，我没有参与选择）
↓
第6页以后：正式量表（感知收益、感知风险、信任、使用意愿等）

---

## 操纵检验的判定标准

建议型组正确答案：MC1选A或B，MC2选A
委托型组正确答案：MC1选C或D，MC2选B

如果被试答错操纵检验题，说明操纵失败，该被试数据需要在分析时排除。
通常操纵失败率低于15%，否则需要检查操纵材料是否足够清晰。

## 如需修改完成码

打开 advisory.html，搜索 "ADV-2024" 替换为您想要的码。
打开 agentic.html，搜索 "AGT-2024" 替换为您想要的码。
建议使用有意义的码，如：
- 建议型：ADVISE-01
- 委托型：AGENT-01

---

## 如需修改商品内容

在 advisory.html 中找到 const PRODUCTS = [...] 那段代码，
修改 name（商品名）、price（价格）、store（门店名）即可。
同时在下方三个 .product-card 的HTML中同步修改标签和描述文字。

在 agentic.html 中，AI选定的商品信息在 .chosen-product 区块内，
搜索"桂花乌龙拿铁"可以找到所有需要修改的地方。
